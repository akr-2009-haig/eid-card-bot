🎉 تصميم بوت بطاقات تهنئة عيد الفطر

فكرة البوت

بوت تيليجرام يقوم بإنشاء بطاقة تهنئة عيد الفطر تلقائيًا.

طريقة العمل:

1️⃣ المستخدم يدخل البوت
2️⃣ يشترك في القنوات الإجبارية
3️⃣ يرسل اسمه
4️⃣ البوت يختار قالب من القوالب
5️⃣ يضيف النص داخل الصورة:

تهنئة من
محمد

6️⃣ يرسل الصورة للمستخدم بصيغة PNG
7️⃣ يرسل نص وأزرار يحددها الأدمن تحت الصورة.
